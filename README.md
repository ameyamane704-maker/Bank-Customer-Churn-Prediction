# Bank Customer Churn Prediction

## Project Overview
This project predicts whether a bank customer is likely to leave the bank using Machine Learning.

## Dataset
European Bank Customer Dataset

Total Records: 10,000

## Objective
Predict customer churn and identify high-risk customers.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Streamlit

## Machine Learning Model
Random Forest Classifier

Accuracy: 86.5%

## Key Findings
- Age is the strongest churn predictor.
- Customers with 3-4 products churn more.
- Inactive customers are more likely to leave.
- Germany shows higher churn rates.
